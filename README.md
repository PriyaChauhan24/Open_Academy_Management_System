# Open Academy Management System

## Description
This is a Django-based Open Academy Management System to manage courses, students, and instructors.

## Features
- Course Management
- Student Registration
- Trainer Assignment

## Technologies Used
- Python
- Django
- HTML/CSS

## How to Run
pip install -r requirements.txt
python manage.py runserver

## Setup Database
python manage.py migrate
python manage.py createsuperuser
